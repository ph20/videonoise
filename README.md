# Videonise

Videonise is a simple Python script to generate a white noise video with audio.

## Build package

To build the package, run:

```bash
make build
```

## Installation

To install, run:

```bash
make install
```
## Usage
To generate a white noise video, run:
```bash
videonoise -o output.mp4 -w 640 -t 480 -f 30 -d 10
```
